# 3 复读机

原始 URL：`https://loj.ac/p/3`

## 题目描述

输入一些正整数，将它们原样输出。

## 提示

这道题用于测试文件输入输出，请使用文件输入输出而非标准输入输出。

- 输入文件：`copycat.in`
- 输出文件：`copycat.out`

## 输入格式

第一行一个正整数 `T`，表示有 `T` 组测试数据。  
接下来 `T` 行，每行一个正整数 `a`。

## 输出格式

输出 `T` 行，每行一个正整数 `a`。

## 样例 1

输入：

```text
3
1
2
3
```

输出：

```text
1
2
3
```

## 样例 2

输入：

```text
1
1000000000000000000000000000000000
```

输出：

```text
1000000000000000000000000000000000
```

## 数据范围

对于所有测试点，`1 <= T <= 10`，`1 <= a <= 10^1000`。
